## System

| Begriff | Definition |
| --- | --- |

| **VSDS** | Vorgänger-Stammdatensystem von FiANTEC (Vertriebsstammdaten) |

## Fachbegriffe (aus XLSX Begrifferläuterungen)

| Begriff | Definition |
| --- | --- |
| **Factory** | Organisation, die der Vertriebsorganisation Produkte zum Verkauf zur Verfügung stellt (eigene oder fremde Produktgeber) |
| **Factory Person** | Person aus Sicht der Factory — immer der Versicherungsnehmer/Vertragspartner |
| **FactoryVertrag** | Aus einem Antrag hervorgegangener Vertrag mit Vertragsbeteiligten aus Sicht der Factory |
| **Factory Ereignis** | Ereignis in der Factory im Rahmen der Antragsbearbeitung oder Bestandesführung |
| **Portefeuille** | Menge von Referenzen auf Personen und Verträge, die eine Vertriebseinheit zu betreuen hat |
| **Betreuungsportefeuille** | Definiert, welche VE für Betreuung eines Vertrages/Kunden zuständig ist |
| **Gebiet** | Aufteilung der Schweiz für VMA — besteht aus Ortschaften, Strassen, PLZ |
| **Vollmacht/Mandat** | Legitimiert eine Person/VO für Zugriff auf Daten (Verwaltungsmandat, Betreuungsmandat, Analyse-/Auskunftsmandat) |
| **Kunde** | Rechtsgültiger Vertragspartner |
| **Zusammenarbeitsmodelle** | Definieren Teamselling, Vermittlungen und Kooperationspartner-Zusammenarbeit |
| **Fallcode** | Abgeschafft — neu: Abschlussdaten. Spezifische Kombination = Zusammenarbeitsmodell (ZAM) |
| **LeistungsVereinbarung** | Vereinbarung zwischen Kanalsteuerung und Factory über Produkte und Konditionen |
| **Vergütungszuordnung** | Definiert, welche Vertriebseinheit(en) bei provisionsrelevanten Ereignissen zu vergüten ist |
